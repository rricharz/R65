.TH R65 1

.SH NAME
R65 - operating system manual

.SH SYNOPSIS
.PP
This manual describes the
R65 operating system and
its basic functions.

.SH HISTORY
.PP
The R65 was used as a
personal word processor.

.PP
Later the system was
extended with Pascal and
additional software.

.PP
A modern replica keeps the
system alive inside an
emulator environment.

.SH BASIC SOFTWARE MODULES

.PP
The system software consists
of separate modules stored
in EPROM.

.IP R65 CRT Controller
  (E000-E7FF)
  Video and keyboard handling.

.IP R65 I/O Controller
  (E800-EFFF)
  Printer, RS-232, interrupts,
  audio tape routines.
  Tape I/O not emulated.

.IP R65 Floppy Controller
  (F000-F7FF)
  Two drives, sequential I/O.

.IP R65 System Monitor
  (F800-FFFF)
  Machine code monitor and
  utilities.

.SH STOP QUIT AND SHUTDOWN

.PP
The emulator recognizes
special keys and executes
them immediately.

.IP SHIFT-ESC  STOP
  Emulate a 6502 NMI break.

.IP SHIFT-MENU QUIT
  Quit the R65 emulator.

.IP SHIFT-ALT SHUTDOWN
  Quit emulator and shut down
  Linux.

.PP
Open sequential files are
not closed when these keys
are used.
