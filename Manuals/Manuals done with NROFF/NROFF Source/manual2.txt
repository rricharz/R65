.TH R65 1

.SH INSTALLATION

.PP
The R65 emulator can be
downloaded from GitHub and
built on the local system.

.SS Build and install
.PP
Typical build steps:

.nf
cd
git clone --depth 1 \
git://github.com/rricharz/R65
sudo apt-get install \
libgtk-3-dev
cd R65
cp R65-Emulator.desktop \
~/desktop
cd R65-Emulator
make clean; make
mkdir Files
.fi

.SS Mandatory fonts
.PP
The emulator needs the
original R65 fonts.

.PP
Install the fonts on the
host system and restart the
emulator afterwards.

.SS Add additional disks
.PP
Disk images are stored in
the Files directory.

.PP
Copy disk image files into
Files and select them in the
emulator.

.SS Upgrade the emulator
.PP
To upgrade, pull the latest
sources and rebuild.

.nf
cd R65
git pull
cd R65-Emulator
make clean; make
.fi

.SH EMULATOR FRONT PANEL

.PP
The front panel shows a
KIM-1 style LED display plus
status indicators.

.SS Buttons
.PP
Three buttons are on the top
right of the window.

.IP BREAK
  Execute a 6502 NMI break.

.IP QUIT
  Exit the emulator.

.IP SHUTDOWN
  Exit and shut down Linux.

.PP
Keyboard shortcuts:

.IP SHIFT-ESC
  Same as BREAK.

.IP SHIFT-MENU
  Same as QUIT.

.IP SHIFT-ALT
  Same as SHUTDOWN.

.SS Seven segment displays
.PP
The KIM-1 display is driven
by user programs.

.PP
If unused, the emulator may
show host status values.

.IP "6502 PC and S"
  Shows program counter and
  stack pointer in hex.

.IP "Pascal PC and free"
  Shows Pascal PC (decimal)
  and free pages (hex).

.SS Drive indicators
.PP
Two LEDs glow while drives
are spinning.

.PP
Names of mounted floppies
are shown in dot matrix
displays.

.SH MONITOR COMMANDS

.PP
The monitor uses short
commands and parameters.

.SS Abbreviations
.nf
exp1   hex expression
exp2   hex expression
filnam file name (max 16)
wfilnam file name with * ?
drive  0 or 1
cy     cyclus number
[ ]    optional argument
.fi

.PP
Below is a compact command
reference.

.SS File commands

.IP COPY
  COPY filnam[.cy],fd,td
  Copy file from drive fd to
  drive td.

.IP DELETE
  DELETE filnam[.cy][,d]
  Delete file on drive d.

.IP DIR
  DIR drive
  Directory listing.

.IP NEW
  NEW filnam[.cy][,d]
  Create a new empty file.

.IP RENAME
  RENAME filnam[.cy][,d]
  Rename a file.

.IP PROTECT
  PROTECT filnam[.cy][,d]
  Protect a file.

.IP PACK
  PACK [drive]
  Pack disk space.

.SS Floppy commands

.IP FLOPPY
  FLOPPY disknam[,drive]
  Select a floppy image.

.IP FDIR
  FDIR [drive]
  Floppy directory.

.IP DFORMAT
  DFORMAT diskname,drive
  Format a floppy disk.

.SS Import and export

.IP EXPORT
  EXPORT filnam[.cy][,d]
  Export file to host.

.IP IMPORT
  IMPORT filnam[,d]
  Import file from host.

.SS Memory and tools

.IP DIS
  DIS [exp1][,exp2]
  Disassemble memory range.

.IP EDIT
  EDIT filnam[.cy][,d]
  Edit a file.

.IP LOAD
  LOAD filnam[.cy][,d][,a]
  Load file to memory.

.SS Run and debug

.IP GO
  GO [exp1][,regs]
  Run code at address.

.IP GSB
  GSB [exp1][,regs]
  Run with subroutine call.

.IP REG
  REG [regs]
  Show or set registers.

.IP CLRB
  CLRB exp1
  Clear breakpoint.

.SS Date and printing

.IP DATE
  Show and edit the date.

.IP PRTB
  Print a memory block.
