.TH NROFF 1

.SH NAME
NROFF - text formatter for R65

.SH SYNOPSIS
.PP
NROFF filnam[:sub[.drv]] [linewidth]

.PP
Default line width is 40.

.SH DESCRIPTION
.PP
NROFF formats plain text
files using simple dot
commands. Lines starting with a dot
are formatting requests.All other lines
are normal text and are formatted using
fill mode.

.SH TEXT MODES

.IP Fill mode
  Words are wrapped to fit
  the current line width.

.IP No fill mode
  Text is printed exactly as
  entered.

.IP .fi
  Enable fill mode.

.IP .nf
  Enable no fill mode.

.SH HEADINGS

.IP .TH
  Manual header line.

.IP .SH
  Section heading.
  Uses underline or inverse
  video depending on width.

.IP .SS
  Subsection heading.

.SH PARAGRAPHS

.IP .PP
  Start new paragraph.

.IP .br
  Line break.

.IP .sp [n]
  Insert blank lines.

.SH INDENTATION

.IP .RS
  Increase indentation.

.IP .RE
  Restore indentation.

.IP .IP label
  Indented paragraph.
  Label followed by text
  with hanging indent.

.SH TABS

.IP .ta n1 n2 ...
  Set tab stops.

.PP
Tabs are expanded during
output.

.PP
If no tabs are defined,
default spacing is 8 cols.

.SH TEXT STYLE

.IP .B text
  Bold line.
  On 40 columns inverse
  video is used.

.SH INPUT RULES

.PP
Maximum input line length
depends on buffer size.

.PP
Control characters except
TAB are ignored.

.SH OUTPUT WIDTH

.PP
Line width may be given as
optional parameter.

.IP Example
  NROFF MANUAL:B,1 80

.PP
Widths above 40 use printer
style headings.

.SH TABLES

.PP
Tables are created using
no fill mode and tabs.

.PP
Example:
.ta 10 20 30
.nf
COL1    COL2    COL3
====	====	====
AA      BBB     CCCC
.fi

.SH LIMITATIONS

.PP
No macros or registers.

.PP
No escape sequences.

.PP
Formatting is single pass.
