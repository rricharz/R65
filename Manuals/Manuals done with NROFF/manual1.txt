NAME
====
R65 - operating system manual

SYNOPSIS
========

This manual describes the R65 operating system and its basic functions.

HISTORY
=======

The R65 was used as a personal word processor.

Later the system was extended with Pascal and additional software.

A modern replica keeps the system alive inside an emulator environment.

BASIC SOFTWARE MODULES
======================

The system software consists of separate modules stored in EPROM.
R65 CRT Controller(E000-E7FF) Video and keyboard handling.
R65 I/O Controller(E800-EFFF) Printer, RS-232, interrupts, audio tape routines. Tape I/O
not emulated.
R65 Floppy Controller(F000-F7FF) Two drives, sequential I/O.
R65 System Monitor(F800-FFFF) Machine code monitor and utilities.

STOP QUIT AND SHUTDOWN
======================

The emulator recognizes special keys and executes them immediately.
SHIFT-ESC  STOPEmulate a 6502 NMI break.
SHIFT-MENU QUITQuit the R65 emulator.
SHIFT-ALT SHUTDOWNQuit emulator and shut down Linux.

Open sequential files are not closed when these keys are used.
