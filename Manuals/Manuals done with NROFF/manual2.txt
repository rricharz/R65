INSTALLATION
============

The R65 emulator can be downloaded from GitHub and built on the local system.

BUILD AND INSTALL
-----------------

Typical build steps:
cd
git clone --depth 1 \
git://github.com/rricharz/R65
sudo apt-get install \
libgtk-3-dev
cd R65
cp R65-Emulator.desktop \
~/desktop
cd R65-Emulator
make clean; make
mkdir Files

MANDATORY FONTS
---------------

The emulator needs the original R65 fonts.

Install the fonts on the host system and restart the emulator afterwards.

ADD ADDITIONAL DISKS
--------------------

Disk images are stored in the Files directory.

Copy disk image files into Files and select them in the emulator.

UPGRADE THE EMULATOR
--------------------

To upgrade, pull the latest sources and rebuild.
cd R65
git pull
cd R65-Emulator
make clean; make

EMULATOR FRONT PANEL
====================

The front panel shows a KIM-1 style LED display plus status indicators.

BUTTONS
-------

Three buttons are on the top right of the window.
BREAK   Execute a 6502 NMI break.
QUIT    Exit the emulator.
SHUTDOWNExit and shut down Linux.

Keyboard shortcuts:
SHIFT-ESCSame as BREAK.
SHIFT-MENUSame as QUIT.
SHIFT-ALTSame as SHUTDOWN.

SEVEN SEGMENT DISPLAYS
----------------------

The KIM-1 display is driven by user programs.

If unused, the emulator may show host status values.
"6502 PC and S"Shows program counter and stack pointer in hex.
"Pascal PC and free"Shows Pascal PC (decimal) and free pages (hex).

DRIVE INDICATORS
----------------

Two LEDs glow while drives are spinning.

Names of mounted floppies are shown in dot matrix displays.

MONITOR COMMANDS
================

The monitor uses short commands and parameters.

ABBREVIATIONS
-------------
exp1   hex expression
exp2   hex expression
filnam file name (max 16)
wfilnam file name with * ?
drive  0 or 1
cy     cyclus number
[ ]    optional argument

Below is a compact command reference.

FILE COMMANDS
-------------
COPY    COPY filnam[.cy],fd,td Copy file from drive fd to drive td.
DELETE  DELETE filnam[.cy][,d] Delete file on drive d.
DIR     DIR drive Directory listing.
NEW     NEW filnam[.cy][,d] Create a new empty file.
RENAME  RENAME filnam[.cy][,d] Rename a file.
PROTECT PROTECT filnam[.cy][,d] Protect a file.
PACK    PACK [drive] Pack disk space.

FLOPPY COMMANDS
---------------
FLOPPY  FLOPPY disknam[,drive] Select a floppy image.
FDIR    FDIR [drive] Floppy directory.
DFORMAT DFORMAT diskname,drive Format a floppy disk.

IMPORT AND EXPORT
-----------------
EXPORT  EXPORT filnam[.cy][,d] Export file to host.
IMPORT  IMPORT filnam[,d] Import file from host.

MEMORY AND TOOLS
----------------
DIS     DIS [exp1][,exp2] Disassemble memory range.
EDIT    EDIT filnam[.cy][,d] Edit a file.
LOAD    LOAD filnam[.cy][,d][,a] Load file to memory.

RUN AND DEBUG
-------------
GO      GO [exp1][,regs] Run code at address.
GSB     GSB [exp1][,regs] Run with subroutine call.
REG     REG [regs] Show or set registers.
CLRB    CLRB exp1 Clear breakpoint.

DATE AND PRINTING
-----------------
DATE    Show and edit the date.
PRTB    Print a memory block.
